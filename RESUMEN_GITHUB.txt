🚀 CÓMO EMPEZAR (5 MINUTOS)
════════════════════════════════════════════════════════════════════════════════

1️⃣ CLONAR
  git clone https://github.com/Gutihope/Aplicacion_contable.git
  cd Aplicacion_contable

2️⃣ INICIA POSTGRES
  docker-compose up -d postgres

3️⃣ INSTALA Y CARGA PUC
  cd backend
  npm install
  npm run db:push
  npm run seed:puc "C:\Users\...\Puc aplicación.xlsx"
  # Responde: sí

4️⃣ INICIA BACKEND
  npm run dev
  # Espera a ver: "🚀 Servidor iniciado en puerto 5000"

5️⃣ INICIA FRONTEND (NUEVA TERMINAL)
  cd frontend
  npm install
  npm run dev
  # Espera a ver: "➜  Local:   http://localhost:5173/"

6️⃣ ACCEDE
  Frontend: http://localhost:5173
  API:      http://localhost:5000/api/v1
  BD GUI:   http://localhost:5555
